Descripción

Este proyecto fue realizado para practicar el uso de constructores (__init__) y destructores (__del__) en Python.
El programa está organizado usando una estructura de modelos y servicios, como se trabajó en clase.

Estructura del proyecto

modelos/: contiene la clase Archivo, donde se usan el constructor y el destructor.

servicios/: contiene la clase que maneja las acciones sobre el archivo.

main.py: archivo principal desde donde se ejecuta el programa.

Cómo ejecutar el programa

Para ejecutar el programa, ubíquese en la carpeta del proyecto y ejecute el siguiente comando:

python main.py

Uso de __init__ y __del__

El constructor (__init__) se utiliza para abrir el archivo al momento de crear el objeto.

El destructor (__del__) se ejecuta cuando el programa finaliza y se encarga de cerrar el archivo.
